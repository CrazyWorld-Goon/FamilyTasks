# Брендбук: «Дом и задачи»

Документ для разработчиков: визуальный язык веб-приложения, токены и практика внедрения. Источник правды по стилю — `src/App.css` (CSS-переменные) и `src/constants.ts` (персональные цвета семьи).

## 1. Позиционирование и тон

- **Название в интерфейсе:** «Дом и задачи».
- **Субтитул в шапке:** «Семья · питомцы · покупки».
- **Настроение UI:** тёплый «домашний» фон, мягкие тени, округлые формы, без агрессивного контраста; дружелюбные подсказки, не сухой продуктовый интерфейс.
- **Язык:** русский; даты/время через `toLocaleString("ru-RU", …)` где уместно.

## 2. Типографика

| Элемент            | Параметры |
|--------------------|-----------|
| Семейство          | **Manrope** (400, 500, 600, 700), подключение в `index.html` с Google Fonts |
| Основной текст     | `body`: размер по умолчанию браузера, `line-height: 1.5`, цвет `--ink` |
| Заголовок бренда   | `.brand h1`: `1.35rem`, `font-weight: 700`, `letter-spacing: -0.02em` |
| Подзаголовок       | `.brand p`: `0.85rem`, цвет `--muted` |
| Заголовки карточек | `.card h2`: `1rem`, `font-weight: 700` |
| Подсказки секций  | `.section-hint`: `0.78rem`, `font-weight: 500`, цвет `--muted` |
| Метаданные строк  | `.row-meta`: `0.78rem`, цвет `--muted` |

**Правило:** не вводить второй шрифт в интерфейсе без согласования; для чисел и времени — те же веса Manrope.

## 3. Цветовая палитра (CSS-переменные)

Все глобальные цвета задаются в `:root` в `App.css`. Новые экраны должны использовать переменные, а не «сырые» hex, кроме редких градиентов кнопок (там зафиксировано сочетание зелёного — см. `.btn-primary`).

### Фон и поверхности

| Переменная       | Назначение |
|------------------|------------|
| `--bg`           | Фон страницы (`#f8f0e6`) + радиальные блики в `body` |
| `--bg-card`      | Карточки, пилюля фазы дня, вторичные блоки |
| `--bg-elevated`  | Поля ввода, плитки с белым дном |

### Текст и границы

| Переменная | Назначение |
|------------|------------|
| `--ink`    | Основной текст |
| `--muted`  | Вторичный текст, подсказки |
| `--line`   | Бордеры карточек, инпутов, неактивных границ |

### Акценты

| Переменная   | Назначение |
|--------------|------------|
| `--accent`   | Розовый бренд; фокус `:focus-visible`, важные акценты |
| `--accent-2` | Садово-зелёный, созвучен с CTA (градиент `.btn-primary`) |
| `--sun`      | «Солнечные» пятна в иллюстрациях/логотипе, тёплые вкрапления |
| `--violet-tabs` | Акцент границы у вкладки «Все» (выбранная) |

### Семантика в списках

| Переменная   | Назначение |
|--------------|------------|
| `--icon-shop` | Иконка корзины в строке покупок |
| `--icon-pet`  | Иконка лапы в задачах питомцев |

Классы: `.row-icon--shop`, `.row-icon--pet` (не дублировать цвет в `style` в React).

### Прочее

- `--link` — цвет текста-ссылки/ «linkish» кнопок в футере.
- `--shadow`, `--shadow-soft` — тени для карточек и мелких контролов.
- `--z-toast` — слой тоста.

## 4. Радиусы и отступы

| Переменная      | Значение | Где |
|-----------------|----------|-----|
| `--radius`      | `16px`   | Карточки `.card` |
| `--radius-sm`   | `12px`   | Кнопки, инпуты, плитки обзора |
| `--radius-pill` | `999px`  | Табы, `resume-link`, `phase-pill` |

- Контейнер: `.app-shell`, `max-width: 920px`, отступы снизу с `env(safe-area-inset-bottom)`.
- Сетка обзора: `.grid-overview` — 1 колонка; с `min-width: 560px` — 2 колонки.

## 5. Компоненты (кратко)

- **Шапка:** логотип `public/images/logo-mark.svg` (48×48), иллюстрация дома `.hero-art` с `min-width: 720px`, блок фазы `.phase-pill` с `IconClock`.
- **Вкладки:** `.tab`, активная — светлый градиент и цветная граница (для «Все» — `--violet-tabs`, для персоны — цвет из `MEMBERS` через inline `borderColor`, это осознанное исключение).
- **Карточка:** `.card` — бордер `--line`, тень `--shadow`.
- **Строка:** `.row` / `.row-title` / `.row-meta` / `.row-actions`.
- **Кнопки:** `.btn-primary` (зелёный градиент), `.btn-warn` (мягкое предупреждение), `disabled` с пониженной `opacity`.
- **Бейджи:** `.badge`, `.badge-feed-window`, `.badge-soft`.
- **Тост:** `.toast` внизу по центру, тёмный фон `#3d3428`, белый текст.
- **Формы:** `.forms`, `.input-row` — flex-wrap, минимальная высота тач-таргетов `2.5rem` на кнопках и полях.

## 6. Персональные цвета семьи

В `src/constants.ts` массив `MEMBERS`: у каждого `id` задано поле `color` (hex). Используется для:

- левой границы плитки в обзоре;
- границы выбранной вкладки персона;
- левого бордера в карточке профиля.

При добавлении члена семьи **сохраняйте** палитру в том же духе: пастель/мягкие насыщенные, не кислотные, хорошо читаемые на `--bg-card`.

## 7. Иконки

- Реализация: `src/components/Icons.tsx` (stroke `currentColor`, кроме лапы).
- **Размер в UI:** чаще `16px` в строках и кнопках, `18px` в заголовках `h2` карточек.
- Не встраивать inline SVG-копии — расширять `Icons.tsx`.

## 8. Ассеты

- Логотип: `public/images/logo-mark.svg`
- Дом: `public/images/house.svg`
- Favicon: `public/favicon.svg`
- `theme-color` в `index.html` совпадает с `--bg` (`#f8f0e6`).

## 9. Доступность и движение

- Клавиатурный фокус: розовая обводка `outline` с `outline-offset: 2px` на интерактивных элементах (см. конец `App.css`).
- `prefers-reduced-motion: reduce` — анимация тоста и микротрансформы кнопок/плиток отключаются.

## 10. Чеклист для новой фичи

1. Цвета — из `:root`; семантика списков — `.row-icon--*`.
2. Отступы и радиусы — из таблицы §4.
3. Шрифт — только Manrope через `--font`.
4. Новые кнопки — модификаторы `.btn*`, не произвольные `background` в компоненте.
5. Иконка — компонент из `Icons.tsx` с осмысленным `className`/`size`.

После изменений визуала обновляйте этот файл, если меняются токены, правила или ассеты.
